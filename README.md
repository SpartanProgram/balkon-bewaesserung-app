# 🌿 Balkonbewässerung Mobile App (Flutter)

A smart mobile app developed with **Flutter & Dart** to control an **automated balcony plant watering system** via IoT hardware. This project is part of a university coursework and demonstrates the use of mobile UI, API integration, and real-time communication with an embedded watering system.

---

## 🚀 Features

- 📱 Intuitive mobile UI built with Flutter
- 💧 Real-time soil moisture monitoring
- ☀️ Manual and automatic watering modes
- 🔄 Live sync between mobile app and ESP32 microcontroller
- 📡 Communication with backend/IoT device over Wi-Fi or Bluetooth
- 🔔 Future feature: Notifications/reminders for low moisture

---

## 🛠️ Tech Stack

- **Frontend**: Flutter & Dart
- **State Management**: Provider (or Riverpod/BLoC if used)
- **IoT**: ESP32 Microcontroller, moisture sensor
- **Communication**: HTTP, REST API or local network calls
- **Design**: Figma mockups
